# 🇯🇵 Metafy Guide Translator

**Metafyのゲームガイドを日本語に自動翻訳するChrome拡張機能です。**

> Flesh and Bloodなどの英語ガイドを、ボタン一発で日本語に翻訳できます。なんとカード名は英語のまま表示される優れもの！1ページあたりの翻訳に数十秒かかるものの実際の記事でカード画像を見ながら翻訳された記事を楽しむ事が出来ます。
> 翻訳結果はキャッシュされるので、同じページは2回目以降タダで読めます。

---

## 📸 使用イメージ

右下の「🇯🇵 翻訳」ボタンを押すだけ！

```
翻訳前: "I view the deck as having a 56 card main deck..."
翻訳後: "私はデッキを56枚のメインデッキとして考えています..."
```

## 🔧 セットアップ手順（全部で約15分）

### STEP 1：ファイルをダウンロードする

1. このページの右上にある緑色の「**Code**」ボタンをクリック
2. 「**Download ZIP**」をクリック
3. ダウンロードされたZIPファイルを**右クリック→「すべて展開」**で解凍する
4. `metafy-extension` というフォルダができればOK

---

### STEP 2：Chromeに拡張機能を読み込む

1. Chromeを開いて、アドレスバーに以下を入力してEnter：
   ```
   chrome://extensions/
   ```

2. 右上の「**デベロッパーモード**」をオンにする（スイッチが青くなればOK）

   ![デベロッパーモードの場所：右上のトグルスイッチ]

3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック

4. さきほど解凍した「**metafy-extension**」フォルダを選択する
   > ⚠️ フォルダの中に `manifest.json` というファイルが見えていればOK

5. 拡張機能の一覧に「**Metafy Guide Translator 🇯🇵**」が表示されれば成功！

---

### STEP 3：Anthropic APIキーを取得する

> APIキーとは、翻訳機能を使うための「鍵」です。無料で取得できますが、使った分だけ少額の料金がかかります（1ページあたり約0.5〜2円）。

#### 3-1. アカウントを作る

1. https://console.anthropic.com にアクセス
2. 「**Sign up**」をクリック
3. メールアドレスとパスワードを入力して登録
4. 届いた確認メールのリンクをクリック

#### 3-2. クレジットカードを登録してチャージする

1. ログイン後、左メニューの「**Billing**」をクリック
2. 「**Add payment method**」でクレジットカードを登録
3. 「**Buy credits**」で最低金額（$5 ＝ 約750円）をチャージ
   > 💡 $5で数百ページ翻訳できます。使い切っても自動請求はされません。

#### 3-3. APIキーを作成する

1. 左メニューの「**API Keys**」をクリック
2. 「**Create Key**」をクリック
3. 名前は何でもOK（例：`metafy`）
4. 表示された `sk-ant-api03-...` という文字列をコピー
   > ⚠️ このキーは**この画面でしか表示されません**！必ずコピーしてください。

---

### STEP 4：APIキーを拡張機能に登録する

1. Chromeの右上にある拡張機能アイコン（🧩のマーク）をクリック
2. 「**Metafy Guide Translator**」をクリック
3. 表示されたポップアップに、さきほどコピーしたAPIキーを貼り付け
4. 「**保存する**」をクリック

これでセットアップ完了です！🎉

## 🕹️ 使い方

1. Metafy（https://metafy.gg）のガイドページを開く
2. ページ右下に「**🇯🇵 翻訳**」ボタンが表示される
3. クリックすると翻訳が始まる（初回は少し時間がかかります）
4. もう一度クリックすると英語に戻る

※1ページごとの翻訳となります。次ページの閲覧や翻訳の際にはもう一度右下のボタンをクリックして原文に戻してから進めて下さい

### 💾 キャッシュ機能について

- 一度翻訳したページは自動的に保存されます
- 次回以降は「**💾 翻訳（保存済）**」と表示され、無料で即座に日本語表示できます
- キャッシュを削除したいときは拡張機能アイコンをクリックして「削除」ボタンを押してください

## ❓ よくある質問

**Q. 翻訳ボタンが表示されません**
→ Metafyのガイドページ（`metafy.gg/guides/...`）でのみ表示されます。

**Q. 「APIキーを設定してください」と表示される**
→ STEP 4の手順でAPIキーを登録してください。

**Q. 翻訳が途中で止まった**
→ APIクレジットが不足している可能性があります。Anthropic Consoleで残高を確認してください。

**Q. 拡張機能を更新したい**
→ 新しい `content.js` などのファイルをフォルダに上書きして、`chrome://extensions/` の🔄ボタンを押してください。

**Q. 翻訳後画面左側に文字が縦に羅列されている、次ページに進むと前ページの翻訳が表示されたままになる**
→ 既知のバグですが、翻訳機能を使用する上で問題無いものですので現状では仕様となります。改善まで今しばらくお待ちください。
また次ページへ進んだ際に前ページの翻訳が表示されたままという問題に関しては
1.再度翻訳ボタンを押し原文に戻す
2.次ページで翻訳ボタンを再度押す
上記手順で使用可能です。

**Q. 翻訳文章をコピペしたいが出来ない**
現状では仕様です。コピペ出来るように対応させる可能性はありますが、基本的には翻訳を使用したPCやブラウザ上でお楽しみ下さい。

## 📝 注意事項

- 著者の意図が分かるクオリティでの翻訳となっておりますが完璧ではございませんのでご了承下さい。
- カード名は殆どの場合英語で表示されますが、カタカナ表記などになることもございます。
- 赤黄青など色の翻訳については赤・レッド・REDなど一貫性無く表記されます。(プロンプトで改善予定)
- 専門的な略語などが多用されている記事の場合には正しく翻訳が出来ない場合がございます。
- このツールはAnthropicのClaudeAPIを使用しています
- APIキーは自分のブラウザにのみ保存され、外部には送信されません
- Metafyの利用規約の範囲内でご使用ください
- このツールはMetafyのサーバーに負荷をかけていません
- このツールはMetafyのガイドを購入したユーザーが読みやすくするための補助ツールです。コンテンツの再配布や無断転載はしていません
- 元記事、翻訳後の記事の譲渡やSNSアップロードは絶対にしない様にお願いいたします
